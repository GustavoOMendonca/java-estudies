# Java Studies

Repositório dedicado aos meus estudos em  **Java** , contendo exercícios, testes e exemplos desenvolvidos durante o processo de aprendizagem da linguagem.

## Objetivo

Organizar minha evolução em Java através de pequenos programas e experimentos práticos, registrando o aprendizado de forma progressiva.

## 📁 Estrutura

```
src/
 ├── 01-basics/
 ├── 02-conditionals/
 ├── 03-loops/
 └── ...
```

Cada pasta representa um tópico estudado.

## Como executar

Compile um arquivo Java:

```
javac NomeDoArquivo.java
```

Execute:

```
java NomeDoArquivo
```

## Observações

* Este repositório é focado em aprendizado.
* Os códigos podem ser simples ou experimentais.
* A organização evoluirá conforme meus estudos avançarem.
